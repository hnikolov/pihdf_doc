Command 'clean'

Format

module [MPATH] clean

Command clean removes the files, which have been generated as a result of running simulation or conversion to verilog. In particular, this command deletes the following files:

Extension Description
.pyc python bytecode files
.vcd wave-form files from simulations
.v verilog files generated for co-simulation and synthesis
.tvr test-vectors and results from simulations

Command clean deletes also directory coverage_html.
